# ✈ Aircraft Detection using Images and Videos

## 📌 Project Overview
This project is an **AI-powered Aerospace Surveillance System** that automatically detects aircraft in images and videos using **YOLOv8** and **Computer Vision** techniques.  
It improves aviation safety, airport monitoring, and airspace surveillance by reducing manual monitoring efforts.

---

## 🚀 Features
- Real-time aircraft detection in images and videos  
- Powered by **YOLOv8** deep learning model  
- Interactive **Streamlit dashboard** for easy use  
- Detection summary with confidence scores  
- AI-generated textual explanation using GPT model  

---

## ⚙ Technologies Used
- Python  
- OpenCV  
- NumPy  
- Matplotlib  
- PyTorch  
- YOLOv8 (Ultralytics)  
- Streamlit  
- Hugging Face Transformers (GPT/BERT)
